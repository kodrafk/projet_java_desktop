📋 DÉTAILS AFFICHÉS
-------------------

POUR TOUS LES ÉVÉNEMENTS :
  • Heure (🕐 10:00)
  • Badge ACTIF (vert)
  • Titre de l'événement (gras, 16px)
  • Coach (👤 Coach : HAMZA)
  • Lieu (📍 Lieu : PARCOURS STADE MENZAH)
  • Bouton "Voir détails" (bleu)

POUR ÉVÉNEMENTS EXTÉRIEURS :
  • Encadré météo jaune avec :
    - Icône météo (☀️ 🌤️ ⛅ etc.)
    - Description (Principalement dégagé)
    - Température (14/22°C)
  • Message santé : "STAY HYDRATED" (bleu)
  • Bordure bleue (#2196F3)

POUR ÉVÉNEMENTS INTÉRIEURS :
  • Encadré gris avec :
    - Icône 🏢
    - Message "Événement en intérieur"
    - "Pas de météo nécessaire"
  • Bordure orange (#D6A46D)

🚀 COMMENT TESTER
-----------------
1. Compilez : mvn clean compile
2. Lancez : mvn javafx:run
3. Naviguez : Événements → Calendrier avec Météo
4. Sélectionnez une date avec événements
5. Vérifiez l'affichage détaillé !
