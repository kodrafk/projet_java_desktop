# 💬 Guide : Notifications des Messages du Coach

## 🎯 Vue d'ensemble

Votre coach peut maintenant vous envoyer des messages personnalisés qui apparaissent directement dans votre section **Objectives**. Vous recevrez une **notification élégante** pour ne jamais manquer un message important !

---

## ✨ Comment ça marche ?

### 1️⃣ Votre coach vous envoie un message

Depuis son interface admin, votre coach :
- Sélectionne votre profil
- Écrit un message personnalisé
- Peut l'envoyer aussi par SMS (optionnel)
- Clique sur "Send Message"

### 2️⃣ Vous recevez une notification

Quand vous ouvrez la section **⚖️ My Goal & Progress** :

```
╔════════════════════════════════════════════╗
║  👨‍⚕️  Your Coach Wrote to You!            ║
║       You have 2 new messages              ║
║                                            ║
║  💬 Check your messages in the Objectives ║
║     section                                ║
╚════════════════════════════════════════════╝
```

**Caractéristiques** :
- 🎨 Design bleu élégant avec gradient
- 👨‍⚕️ Icône de coach visible
- ⏱️ Disparaît automatiquement après 4 secondes
- 🖱️ Cliquez dessus pour la fermer immédiatement

### 3️⃣ Vous lisez vos messages

Scrollez vers le haut de la page Objectives pour voir :

```
┌─────────────────────────────────────────────┐
│ 💬 Messages from Coach              [2]     │
├─────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────┐ │
│ │ 👨‍⚕️ 26/04 14:30                    📱  │ │
│ │                                         │ │
│ │ Bravo pour vos progrès ! 🎉            │ │
│ │ Vous avez perdu 2 kg cette semaine.    │ │
│ │ Continuez comme ça ! 💪                │ │
│ │                                         │ │
│ │ [✓ Got it]                             │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

**Éléments** :
- 💬 Titre avec badge rouge indiquant le nombre de messages
- 👨‍⚕️ Icône du coach
- 📅 Date et heure du message
- 📱 Badge SMS si envoyé aussi par SMS
- 💬 Contenu du message
- ✓ Bouton "Got it" pour marquer comme lu

### 4️⃣ Vous marquez le message comme lu

Cliquez sur **✓ Got it** :
- ✅ Le message disparaît de la liste
- ✅ Le badge de comptage se met à jour
- ✅ Confirmation "Message marked as read"

---

## 📍 Où trouver les messages ?

### Emplacement

Les messages apparaissent dans la section **⚖️ My Goal & Progress** :

1. Cliquez sur **Home** dans le menu
2. Cliquez sur **My Kitchen** ou naviguez vers Objectives
3. Ouvrez **⚖️ My Goal & Progress**
4. Les messages sont **en haut de la page**, juste après le header

### Ordre d'affichage

De haut en bas :
1. 💬 **Messages from Coach** (si vous avez des messages)
2. 🎯 **Progress** (votre objectif actuel)
3. 📈 **Progress Chart** (graphique)
4. 📸 **Transformation Gallery**
5. 📊 **Log my weight today**
6. 📋 **Weight History**
7. 🎯 **Set my goal**

## 🔔 Quand la notification apparaît-elle ?

### Déclenchement

La notification apparaît **automatiquement** quand :
- ✅ Vous ouvrez la section **Objectives**
- ✅ Vous avez au moins 1 message non lu
- ✅ C'est la première fois que vous ouvrez la section depuis la réception du message

### Fréquence

- **Une seule fois** par message
- Après avoir marqué comme lu, la notification ne réapparaît plus
- Si vous recevez un nouveau message, une nouvelle notification s'affichera

## 💡 Conseils d'utilisation

### Pour ne pas manquer de messages

1. **Consultez régulièrement** la section Objectives
2. **Regardez le badge rouge** sur "Messages from Coach"
3. **Lisez vos SMS** si votre coach envoie aussi par SMS

### Pour gérer vos messages

1. **Lisez immédiatement** les messages importants
2. **Marquez comme lu** après lecture
3. **Gardez votre section propre** en marquant tous les messages lus

### Pour répondre à votre coach

Actuellement, vous ne pouvez pas répondre directement dans l'application.

**Options** :
- 📧 Envoyez un email à votre coach
- 📱 Appelez votre coach
- 💬 Utilisez un autre canal de communication

## ❓ FAQ

### Q: La notification ne s'affiche pas ?

**Vérifiez** :
- Avez-vous des messages non lus ?
- Êtes-vous dans la section Objectives ?
- Avez-vous déjà vu la notification pour ce message ?

### Q: Comment supprimer un message ?

Actuellement, vous ne pouvez que **marquer comme lu**. Les messages lus disparaissent automatiquement de la liste.

### Q: Puis-je voir les anciens messages ?

Non, une fois marqués comme lus, les messages disparaissent. Ils restent dans l'historique de votre coach.

### Q: Puis-je désactiver les notifications ?

Non, les notifications sont essentielles pour ne pas manquer les messages importants de votre coach.

### Q: Le badge SMS signifie quoi ?

Le badge 📱 indique que le message a aussi été envoyé sur votre téléphone par SMS.

### Q: Combien de temps la notification reste-t-elle ?

La notification reste affichée **4 secondes** puis disparaît automatiquement. Vous pouvez aussi cliquer dessus pour la fermer immédiatement.
