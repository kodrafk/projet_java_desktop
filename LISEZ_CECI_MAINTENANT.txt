                              📝 RÉSUMÉ RAPIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Ouvrir http://localhost/phpmyadmin

2. Cliquer sur "nutrilife" à gauche

3. Cliquer sur l'onglet "SQL"

4. Ouvrir le fichier : INSERT_USERS_DIRECT.sql

5. Copier TOUT le contenu (Ctrl+A puis Ctrl+C)

6. Coller dans phpMyAdmin (Ctrl+V)

7. Cliquer "Exécuter"

8. Vous devriez voir : "SUCCESS: 5 users added!"

9. Dans le terminal :
   mvn clean compile
   mvn javafx:run

10. Menu → Personalized Messages → ✅ 5 utilisateurs visibles !
