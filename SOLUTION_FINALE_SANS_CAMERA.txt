═══════════════════════════════════════════════════════════════════════════
    🚀 UTILISATION
═══════════════════════════════════════════════════════════════════════════

ÉTAPE 1 : COMPILEZ LE PROJET
──────────────────────────────────────────────────────────────────────────
Ouvrez un terminal dans projetJAV et tapez :

    mvn clean compile

Attendez que la compilation se termine.


ÉTAPE 2 : LANCEZ L'APPLICATION
──────────────────────────────────────────────────────────────────────────
Double-cliquez sur :

    📁 LAUNCH_APP.bat

Ou tapez :

    mvn javafx:run


ÉTAPE 3 : ENREGISTREZ VOTRE VISAGE
──────────────────────────────────────────────────────────────────────────
1. Connectez-vous avec email/password
2. Menu → Face ID → Enroll (Upload)
3. Cliquez "Choose Photo"
4. Sélectionnez une photo de votre visage
5. Cliquez "Capture"
6. Répétez 3 fois avec 3 photos différentes

✅ "Face ID enrolled successfully!"


ÉTAPE 4 : CONNECTEZ-VOUS AVEC VOTRE VISAGE
──────────────────────────────────────────────────────────────────────────
1. Écran de connexion
2. Cliquez "Face ID - Upload Photo" (nouveau bouton)
3. Cliquez "Choose Photo"
4. Sélectionnez une photo de votre visage
5. Cliquez "Verify Face"

✅ "Welcome, [Votre Nom]! (89% match)"
🎉 Connexion automatique !
