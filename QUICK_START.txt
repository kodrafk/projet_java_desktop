│  📊 VOTRE PROBLÈME                                             │
└────────────────────────────────────────────────────────────────┘

   ❌ MySQL s'arrête immédiatement dans XAMPP
   ❌ phpMyAdmin affiche une erreur de connexion
   ❌ Message: "MySQL shutdown unexpectedly"


┌────────────────────────────────────────────────────────────────┐
│  ✅ CE QUE LE SCRIPT VA FAIRE                                  │
└────────────────────────────────────────────────────────────────┘

   1. Arrêter MySQL
   2. Sauvegarder vos données existantes
   3. Supprimer les fichiers corrompus
   4. Redémarrer MySQL
   5. Créer la base de données nutrilife_db
   6. Créer toutes les tables
   7. Créer le compte admin

   ⏱️  Durée: ~30 secondes


┌────────────────────────────────────────────────────────────────┐
│  🎯 APRÈS LA RÉPARATION                                        │
└────────────────────────────────────────────────────────────────┘

   1. Ouvrez phpMyAdmin: http://localhost/phpmyadmin
   
   2. Vérifiez que la base nutrilife_db existe
   
   3. Lancez votre application Java
   
   4. Connectez-vous avec:
      📧 Email    : admin@nutrilife.com
      🔑 Password : admin123
