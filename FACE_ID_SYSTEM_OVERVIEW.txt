╚══════════════════════════════════════════════════════════════════════════════╝

User clicks "Face ID Login"
    │
    ▼
FaceIDLauncher.launchAuthentication()
    │
    ▼
ProfessionalFaceIDController opens
    │
    ├─► Camera opens (ProfessionalCameraService)
    │
    ├─► Camera preview starts (30 FPS)
    │
    ├─► Face detection loop (every 10 frames)
    │   │
    │   ├─► Face detected? → Show "Face detected"
    │   │
    │   └─► Auto-start authentication after 1s
    │
    ├─► Liveness check
    │   │
    │   ├─► Blink detection
    │   ├─► Head movement
    │   └─► Texture analysis
    │
    ├─► Extract embedding (FaceDetectionService)
    │   │
    │   └─► Python ArcFace → 512D vector
    │
    ├─► Verify with backend (FaceIDService)
    │   │
    │   ├─► Check rate limiting
    │   ├─► Validate liveness
    │   ├─► Compare embeddings
    │   └─► Log attempt
    │
    ├─► Success?
    │   │
    │   ├─► YES → Login user → Close window → onSuccess()
    │   │
    │   └─► NO → Show error → Show retry button
    │
    └─► User clicks "Use Password" → onFailure()
