1. Ajouter des utilisateurs à la base de données :

   a) Ouvrir http://localhost/phpmyadmin
   b) Cliquer sur "nutrilife"
   c) Onglet "SQL"
   d) Copier le contenu de : INSERT_USERS_DIRECT.sql
   e) Coller et cliquer "Exécuter"
   f) Vérifier : "SUCCESS: 5 users added!"


2. Lancer l'application :

   mvn javafx:run


3. Se connecter :

   Email: admin@nutrilife.com
   Password: admin123

                    TEST 1 : USER MANAGEMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Cliquer sur "User Management" dans le menu

2. ✅ Vérifier :
   - Vous voyez "X user(s)" en haut (X > 0)
   - Le tableau affiche tous les utilisateurs
   - Colonnes : Email, Full Name, Role, Status, Created At, Actions

3. Tester la recherche :
   - Taper "john" dans la barre de recherche
   - ✅ Seuls les utilisateurs avec "john" s'affichent

4. Tester le tri :
   - Cliquer sur l'en-tête "Email"
   - ✅ Les utilisateurs sont triés par email

5. Tester les boutons d'action :

   a) Bouton "View" (bleu clair)
      - Cliquer sur "View" pour un utilisateur
      - ✅ Une fenêtre s'ouvre avec les détails
      - Fermer la fenêtre

   b) Bouton "Edit" (bleu)
      - Cliquer sur "Edit" pour un utilisateur
      - ✅ Formulaire d'édition s'ouvre
      - Modifier le prénom
      - Cliquer "Save"
      - ✅ Modification enregistrée

   c) Bouton "Message" (orange)
      - Cliquer sur "Message" pour un utilisateur
      - ✅ Interface de messages s'ouvre
      - Écrire un message de test
      - Cliquer "Send"
      - ✅ Message envoyé

6. Tester "+ Add New User" :
   - Cliquer sur le bouton vert en haut à droite
   - ✅ Formulaire de création s'ouvre
   - Remplir les champs
   - Cliquer "Create"
   - ✅ Nouvel utilisateur créé

1. Cliquer sur "Personalized Messages" dans le menu

2. ✅ Vérifier :
   - Liste des utilisateurs à gauche
   - Badge "👑 Admin" pour les administrateurs
   - Compteur "X users" en haut

3. Sélectionner un utilisateur :
   - Cliquer sur un utilisateur dans la liste
   - ✅ La carte devient verte
   - ✅ "Sending to: [nom]" s'affiche en haut

4. Écrire un message :
   - Taper : "Bonjour, ceci est un message de test."
   - ✅ Compteur de caractères s'affiche
   - Cliquer "Send Message"
   - ✅ Toast de succès
   - ✅ Message apparaît dans l'historique à droite

5. Tester la recherche :
   - Taper un nom dans la barre de recherche
   - ✅ Filtrage en temps réel

6. Tester l'option SMS :
   - Cocher "Send via SMS"
   - Envoyer un message
   - ✅ Badge SMS dans l'historique

   - Catégorie : HEALTH
   - Cliquer "Send Alert"
   - ✅ Toast de succès

   - ✅ Alerte créée avec expiration
