╔══════════════════════════════════════════════════════════════════════════════╗
║                      🚀 DÉMARRAGE EN 3 ÉTAPES                                ║
╚══════════════════════════════════════════════════════════════════════════════╝

  1️⃣  Obtenez votre clé API gratuite
      → https://openweathermap.org/api

  2️⃣  Configurez weather.properties
      → api.key=VOTRE_CLE_ICI

  3️⃣  Lancez l'application
      → mvn javafx:run
