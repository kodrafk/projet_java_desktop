# 🎨 Génération Intelligente d'Images pour Événements

## 🎯 Objectif

Créer un système intelligent qui génère automatiquement des images variées et pertinentes pour chaque événement en fonction de son titre, avec reconnaissance de 15+ catégories de sports et activités.

---

## ✨ Fonctionnalités

### 1. Détection Automatique du Type d'Événement
Le système analyse le titre de l'événement et détecte automatiquement sa catégorie parmi 15+ types différents.

### 2. Prompts Intelligents et Variés
Chaque catégorie dispose de 4 prompts différents pour générer des images variées.

### 3. Cohérence par Titre
Le même titre génère toujours la même image (utilisation du hashCode comme seed).

### 4. Images de Haute Qualité
Tous les prompts incluent des qualificatifs professionnels pour garantir la qualité.

## 🔧 Fonctionnement Technique

### 1. Analyse du Titre
```java
String titreLower = titre.toLowerCase();
```
Le titre est converti en minuscules pour la comparaison.

### 2. Détection de Catégorie
```java
for (Map.Entry<String, String[]> entry : CATEGORIES_PROMPTS.entrySet()) {
    String pattern = entry.getKey();
    String[] keywords = pattern.split("\\|");
    for (String keyword : keywords) {
        if (titreLower.contains(keyword)) {
            // Catégorie trouvée !
        }
    }
}
```

### 3. Sélection du Prompt
```java
int index = Math.abs(titreLower.hashCode()) % prompts.length;
return prompts[index];
```
Utilise le hashCode du titre pour choisir un prompt de manière déterministe.

### 4. Génération de l'URL
```java
String prompt = detecterEtGenererPrompt(titreLower);
prompt += " professional high quality 4k photography realistic lighting";
String encodedPrompt = URLEncoder.encode(prompt, StandardCharsets.UTF_8);
int seed = Math.abs(titre.hashCode()) % 10000;
return "https://image.pollinations.ai/prompt/" + encodedPrompt + 
       "?width=800&height=600&nologo=true&seed=" + seed;
```
