┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  🎯 CONCEPT                                                                  │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

  Le système détecte AUTOMATIQUEMENT si un événement est:
  
  🌤️  EN PLEIN AIR    → Météo affichée ✅
  🏢  EN INTÉRIEUR    → Pas de météo ❌
  
  
  AVANTAGES:
  
  ✅ Pas de météo inutile pour les événements en salle
  ✅ Économie d'appels API (60% en moins!)
  ✅ Interface plus claire
  ✅ Détection automatique intelligente


╔══════════════════════════════════════════════════════════════════════════════╗
║                        🔍 COMMENT ÇA MARCHE?                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

  Le système analyse 3 éléments:
  
  1️⃣  LIEU de l'événement
  2️⃣  NOM de l'événement
  3️⃣  DESCRIPTION de l'événement
  
  
  Il cherche des MOTS-CLÉS pour déterminer le type:


┌─ MOTS-CLÉS EXTÉRIEUR (plein air) ────────────────────────────────────────────┐
│                                                                              │
│  🌳 Lieux naturels:                                                          │
│     parc, jardin, plage, lac, mer, montagne, forêt                           │
│                                                                              │
│  🏟️  Infrastructures:                                                        │
│     stade, terrain, piste, court, parcours, circuit                          │
│                                                                              │
│  🏃 Activités:                                                               │
│     course, marathon, trail, vélo, football, tennis                          │
│                                                                              │
│  🎪 Événements:                                                              │
│     festival, fête, marché, pique-nique, camping                             │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ MOTS-CLÉS INTÉRIEUR ─────────────────────────────────────────────────────────┐
│                                                                              │

╚══════════════════════════════════════════════════════════════════════════════╝

     → 100 appels API météo

  AVANTAGES:
  
  ✅ Moins de consommation de quota API
  ✅ Application plus rapide
  ✅ Moins de données réseau
  ✅ Meilleure expérience utilisateur
