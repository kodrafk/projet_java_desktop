    🔧 DÉPANNAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════

    ❌ CAMÉRA NOIRE
    ─────────────────────────────────────────────────────────────────────
    Cause : Caméra utilisée par autre app
    
    Solution :
        1. Fermez Skype, Teams, Zoom, Discord
        2. Double-cliquez : FIX_CAMERA_BLACK_SCREEN.bat
        3. Relancez l'application


    ❌ "NO FACE DETECTED"
    ─────────────────────────────────────────────────────────────────────
    Causes : Trop loin, trop sombre, de profil
    
    Solution :
        • Rapprochez-vous (50-70 cm)
        • Allumez une lumière
        • Regardez la caméra


    ❌ "EYES NOT CLEAR"
    ─────────────────────────────────────────────────────────────────────
    Causes : Yeux fermés, lunettes avec reflets, trop loin
    
    Solution :
        • Ouvrez bien les yeux
        • Enlevez lunettes ou ajustez angle
        • Rapprochez-vous


    ❌ "FACE NOT RECOGNIZED"
    ─────────────────────────────────────────────────────────────────────
    Causes : Changement apparence, éclairage différent
    
    Solution :
        • Réenregistrez votre visage (Face ID → Re-enroll)
        • Attendez cadre VERT avant capture
        • Même éclairage qu'à l'enrollment
