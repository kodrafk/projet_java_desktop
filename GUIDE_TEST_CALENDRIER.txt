🚀 COMMENT TESTER
-----------------

ÉTAPE 1 : Compiler le projet
-----------------------------
Ouvrez un terminal et exécutez :
  mvn clean compile

Ou utilisez votre IDE (IntelliJ IDEA, Eclipse, NetBeans) :
  - IntelliJ : Clic droit sur le projet → Build → Rebuild Project
  - Eclipse : Project → Clean → Build
  - NetBeans : Run → Clean and Build Project

ÉTAPE 2 : Lancer l'application
-------------------------------
  mvn javafx:run

Ou depuis votre IDE :
  - Exécutez la classe MainApp.java

ÉTAPE 3 : Tester la navigation
-------------------------------
1. Connectez-vous à l'application
2. Naviguez vers la page "Événements" (menu latéral)
3. Cliquez sur le bouton "📅 Calendrier avec Météo"
4. ✅ Le calendrier devrait maintenant s'afficher correctement !

ÉTAPE 4 : Vérifier les fonctionnalités
---------------------------------------
Dans le calendrier, vous devriez voir :
  ✓ Le calendrier du mois en cours
  ✓ Les jours avec événements marqués d'un badge
  ✓ Les statistiques du mois (en haut à droite)
  ✓ Les événements du jour sélectionné (panneau de droite)
  ✓ La météo UNIQUEMENT pour les événements extérieurs
  ✓ Le bouton "← Retour aux Événements" en haut à gauche

ÉTAPE 5 : Tester la météo intelligente
---------------------------------------
  🌤️ Événements EXTÉRIEURS (avec météo) :
     - Parc, Stade, Plage, Jardin, Montagne, etc.
     → Affichent la météo avec température et icône

  🏢 Événements INTÉRIEURS (sans météo) :
     - Salle, Restaurant, Gym, Bureau, etc.
     → Affichent "(Intérieur)" sans météo

ÉTAPE 6 : Tester le retour
---------------------------
1. Depuis le calendrier, cliquez sur "← Retour aux Événements"
2. ✅ Vous devriez revenir à la page des événements
