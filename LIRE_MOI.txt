📝 UTILISATION FACE ID:

ENREGISTREMENT (première fois):
1. Lancez l'application
2. Inscrivez-vous / Connectez-vous
3. Allez dans Profil
4. Cliquez "Enroll Face ID"
5. Positionnez votre visage
6. Cliquez "Capture"
7. ✅ Face ID enregistré!

CONNEXION:
1. Écran de connexion
2. Cliquez "Face ID Login"
3. Positionnez votre visage
4. ✅ Connexion automatique!

🐛 PROBLÈMES?

❌ "Python server not running"
   → Le serveur Python n'est pas démarré
   → Solution: Utilisez START.bat (il démarre tout)

❌ "No face detected"
   → Améliorez l'éclairage
   → Regardez directement la caméra
   → Rapprochez-vous

❌ "Face not recognized"
   → Vérifiez que vous avez enregistré votre visage
   → Réenregistrez avec un meilleur éclairage
