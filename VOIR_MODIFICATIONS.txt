2. Téléchargez Python 3.10 ou plus récent

2. Exécutez: INSTALL.bat
3. Attendez la fin de l'installation (2-3 minutes)

┌────────────────────────────────────────────────────────────┐
│ ÉTAPE 3: DÉMARRER LE SERVEUR PYTHON                       │
└────────────────────────────────────────────────────────────┘

1. Dans le dossier python_face_server
2. Double-cliquez sur: START_SERVER.bat
3. Vous verrez:
   
   🚀 SIMPLE FACE RECOGNITION SERVER
   Library: opencv
   Starting server on http://localhost:5000

4. LAISSEZ CETTE FENÊTRE OUVERTE!

┌────────────────────────────────────────────────────────────┐
│ ÉTAPE 4: DÉMARRER L'APPLICATION                           │
└────────────────────────────────────────────────────────────┘

1. Ouvrez un AUTRE terminal dans: projetJAV
2. Exécutez: mvn javafx:run
3. L'application s'ouvre
