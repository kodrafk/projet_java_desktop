🚀 INSTALLATION RAPIDE (3 ÉTAPES) :

1. Créer la table user_alerts :
   > CREATE_USER_ALERTS_TABLE.bat

2. Compiler le projet :
   > mvn clean compile

3. Lancer l'application :
   > mvn javafx:run

🎯 UTILISATION IMMÉDIATE :

EN TANT QU'ADMIN :

1. Connexion avec identifiants admin
2. Menu → 🔔 User Alerts
3. Sélectionner un utilisateur
4. Remplir le formulaire :
   - Title : "Bienvenue !"
   - Message : "Nous sommes ravis de vous accueillir"
   - Type : INFO
   - Category : SYSTEM
5. Cliquer "Send Alert"

EN TANT QU'UTILISATEUR :

1. Connexion avec compte utilisateur
2. Widget d'alertes affiché automatiquement
3. Badge [3] indique 3 alertes non lues
4. Cliquer sur une alerte pour voir les détails
5. Actions : Mark as read / Action / Dismiss

🐛 DÉPANNAGE :

Problème : Table user_alerts n'existe pas
Solution : Exécuter CREATE_USER_ALERTS_TABLE.bat

Problème : Menu "User Alerts" invisible
Solution : Vérifier connexion admin, redémarrer l'application

Problème : Alertes non affichées
Solution : Vérifier table existe, appeler alertsWidget.refresh()

Problème : SMS non envoyés
Solution : Vérifier configuration Twilio dans TwilioService.java
