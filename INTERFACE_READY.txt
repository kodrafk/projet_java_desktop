🐛 DÉPANNAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════

Problème : Application ne démarre pas
→ Vérifier Java : java -version
→ Nettoyer : mvn clean compile
→ Relancer : mvn javafx:run

Problème : Menu "HEALTH AI" invisible
→ Vérifier admin_layout.fxml modifié
→ Vérifier AdminLayoutController.java modifié
→ Recompiler : mvn clean compile

Problème : Dashboard vide
→ Cliquer "Lancer Détection"
→ Vérifier base de données : SHOW TABLES LIKE 'health%';
→ Insérer données test : TEST_ANOMALY_DETECTION.bat

Problème : Erreur de compilation
→ Vérifier tous les fichiers Java présents
→ Vérifier pom.xml
→ Nettoyer : mvn clean
