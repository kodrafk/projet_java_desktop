## 🎯 Conseils pour Débloquer les Badges

### 🚀 Démarrage Rapide (Jour 1)
```
✅ Créer un compte          → 🌟 Welcome!
✅ Compléter le profil      → ✅ Profile Ready
✅ Ajouter une photo        → 📸 Say Cheese!
✅ Activer Face ID          → 🔐 Face Unlocked
```

### 📊 Suivi Régulier (Semaine 1)
```
✅ Logger 1 poids           → ⚖️ First Weigh-In
✅ Logger 3 jours de suite  → 🔥 On a Roll
✅ Logger 5 fois            → 📊 Consistent Tracker
✅ Logger 7 jours de suite  → ⚡ Week Warrior
```

### 🎯 Objectifs (Mois 1)
```
✅ Créer un objectif        → 🎯 Goal Setter
✅ Perdre/gagner 3kg        → 📉 3kg Milestone
✅ Perdre/gagner 5kg        → 🎖️ 5kg Achiever
✅ Atteindre 50%            → 🔥 Halfway Hero
```

### 🎮 Engagement (Continu)
```
✅ Gagner 100 points        → 🎮 Arena Rookie
✅ Compléter 1 défi         → 🎯 Challenge Starter
✅ Logger avant 8h          → 🌅 Early Bird
✅ Logger après 22h         → 🦉 Night Owl
```
