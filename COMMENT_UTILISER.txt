🚀 UTILISATION
────────────────────────────────────────────────────────────────

1. Lancez l'application :
   LANCER_APPLICATION.bat

2. Sur l'écran de connexion, cliquez sur :
   "Face ID Login" (bouton bleu avec icône 🔐)

3. La fenêtre Face ID s'ouvre avec :
   • Fond dégradé professionnel
   • Cercle de caméra lumineux
   • Votre caméra s'allume automatiquement
   • Votre visage apparaît dans le cercle

4. Le système détecte automatiquement votre visage

5. Cliquez sur les boutons selon vos besoins :
   • Retry : Réessayer la détection
   • Use Password : Utiliser le mot de passe
   • Cancel : Annuler et fermer
