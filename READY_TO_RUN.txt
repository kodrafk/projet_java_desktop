🐛 SI PROBLÈME
═══════════════════════════════════════════════════════════════════════════

Problème : Application ne démarre pas
→ Vérifier Java : java -version (doit être ≥ 11)
→ Vérifier Maven : mvn -version
→ Nettoyer : mvn clean compile

Problème : Menu "HEALTH AI" invisible
→ Vérifier que vous êtes connecté en ADMIN
→ Recompiler : mvn clean compile
→ Relancer l'application

Problème : Dashboard vide
→ Cliquer sur "🚀 Lancer Détection"
→ Vérifier MySQL : mysql -u root -p
→ Vérifier tables : SHOW TABLES LIKE 'health%';
→ Insérer données test : TEST_ANOMALY_DETECTION.bat

Problème : Erreur de compilation
→ Vérifier pom.xml
→ Nettoyer : mvn clean
→ Recompiler : mvn compile
