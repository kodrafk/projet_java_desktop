================================================================================
                    POURQUOI CA SE BLOQUE?
================================================================================

RAISON #1: Processus Java bloque (80%)
  → L'application ne s'est pas fermee proprement
  → Le processus reste actif
  → La camera reste ouverte

RAISON #2: Teams/Zoom utilise la camera (15%)
  → Teams/Zoom est ouvert
  → Ils bloquent la camera
  → NutriLife ne peut pas y acceder

RAISON #3: Driver camera bloque (5%)
  → Le driver Windows se bloque
  → Besoin de redemarrer

================================================================================
                    DIAGNOSTIC MANUEL
================================================================================

Si vous voulez comprendre ce qui bloque:

ETAPE 1: Ouvrir Gestionnaire des taches
  → Ctrl+Shift+Esc

ETAPE 2: Onglet "Processus"
  → Cherchez: java.exe, javaw.exe
  → Cherchez: python.exe
  → Cherchez: Teams.exe, Zoom.exe
  → Cherchez: chrome.exe, msedge.exe

ETAPE 3: Terminer ces processus
  → Clic droit → Terminer la tache
  → Faites ca pour TOUS

ETAPE 4: Tester camera Windows
  → Ouvrez l'app "Camera" de Windows
  → Si ca marche → Probleme dans NutriLife
  → Si ca ne marche pas → Probleme driver

ETAPE 5: Tester NutriLife
  → Ouvrez NutriLife
  → Essayez Face ID

================================================================================
                    COMMANDES MANUELLES
================================================================================

Si vous preferez taper les commandes:

Ouvrez CMD (Invite de commandes) et tapez:

Tuer Java:
  taskkill /F /IM java.exe /T
  taskkill /F /IM javaw.exe /T

Tuer Python:
  taskkill /F /IM python.exe /T

Tuer Teams:
  taskkill /F /IM Teams.exe /T

Tuer Zoom:
  taskkill /F /IM Zoom.exe /T

Tuer Chrome:
  taskkill /F /IM chrome.exe /T

Tuer TOUT d'un coup:
  taskkill /F /IM java.exe /T & taskkill /F /IM python.exe /T & taskkill /F /IM Teams.exe /T & taskkill /F /IM Zoom.exe /T
