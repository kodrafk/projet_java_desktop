# 📖 Guide: Profil Admin - Édition et Face ID

## ✅ Fonctionnalités Disponibles

Quand l'admin clique sur l'avatar "**A**" en haut à droite, il peut:

### 1. 👤 Voir Ses Coordonnées (Informations)
- **Nom complet**
- **Email**
- **Date de naissance**
- **Âge**
- **Poids et Taille**
- **IMC (BMI)**
- **Date d'inscription**
- **Statut Face ID**
- **Message de bienvenue**

### 2. ✏️ Éditer Son Profil
Bouton: **"Edit Profile"**

Permet de modifier:
- Prénom et Nom
- Date de naissance
- Poids (kg)
- Taille (cm)
- Photo de profil

### 3. 🎭 Gérer Face ID (Caméra)
Bouton: **"🎭 Manage Face ID"**

#### Ajouter Face ID:
1. Cliquer sur "🎭 Manage Face ID"
2. Si pas encore enregistré, cliquer sur "📷 Enroll Face ID"
3. La caméra s'ouvre automatiquement
4. Suivre les instructions pour capturer 3 positions:
   - Face de face
   - Tourner légèrement à gauche
   - Tourner légèrement à droite
5. Face ID enregistré avec succès!

#### Supprimer Face ID:
1. Cliquer sur "🎭 Manage Face ID"
2. Si déjà enregistré, cliquer sur "🗑 Remove Face ID"
3. Confirmer la suppression
4. Face ID supprimé!

### 4. 🔑 Changer le Mot de Passe
Bouton: **"Change Password"**

Permet de changer le mot de passe de connexion.

### 5. ✏ Modifier le Message de Bienvenue
Bouton: **"✏ Edit Welcome Message"**

Personnaliser le message affiché sur le profil.

## 🚀 Comment Utiliser

### Étape 1: Ouvrir le Profil Admin
```
1. Connectez-vous en tant qu'admin
   Email: admin@nutrilife.com
   Mot de passe: admin123

2. Cliquez sur l'avatar "A" en haut à droite
   (à côté de "ADMINISTRATOR PROFILE")

3. Le profil s'ouvre avec toutes les informations
```

### Étape 2: Éditer les Informations
```
1. Cliquer sur "Edit Profile" (bouton vert)
2. Modifier les champs souhaités
3. Cliquer sur "Save" pour enregistrer
```

### Étape 3: Ajouter Face ID
```
1. Cliquer sur "🎭 Manage Face ID" (bouton bleu)
2. Cliquer sur "📷 Enroll Face ID"
3. Autoriser l'accès à la caméra si demandé
4. Suivre les instructions à l'écran:
   - Position 1: Face de face
   - Position 2: Tourner à gauche
   - Position 3: Tourner à droite
5. Attendre la confirmation
6. Face ID enregistré!
```

### Étape 4: Supprimer Face ID
```
1. Cliquer sur "🎭 Manage Face ID"
2. Cliquer sur "🗑 Remove Face ID" (bouton rouge)
3. Confirmer en cliquant "Yes, Remove"
4. Face ID supprimé!
```

## 🔒 Sécurité Face ID

### Comment ça marche:
- **3 positions** de votre visage sont capturées
- **Chiffrement AES-256-GCM** pour sécuriser les données
- **Stockage local** uniquement (ne quitte jamais l'appareil)
- **Vérification d'identité** rapide et sécurisée

### Avantages:
- ✅ Connexion rapide sans mot de passe
- ✅ Sécurité biométrique
- ✅ Données chiffrées
- ✅ Facile à utiliser
