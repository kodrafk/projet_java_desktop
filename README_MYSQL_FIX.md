# 🚀 CORRECTION RAPIDE - ERREUR MYSQL

## ⚡ SOLUTION EN 1 CLIC

**Double-cliquez sur ce fichier :**

```
MYSQL_MANAGER.bat
```

Puis choisissez l'option **[1]** pour réparer MySQL automatiquement.

---

## 📋 CE QUI VA SE PASSER

Le script va :
1. ✅ Arrêter MySQL
2. ✅ Sauvegarder vos données
3. ✅ Supprimer les fichiers corrompus
4. ✅ Redémarrer MySQL
5. ✅ Créer la base `nutrilife_db`
6. ✅ Créer les tables
7. ✅ Créer le compte admin

**Durée : ~30 secondes**

---

## 🎯 APRÈS LA RÉPARATION

### 1. Vérifier phpMyAdmin
Ouvrez : http://localhost/phpmyadmin

Vous devriez voir la base `nutrilife_db`

### 2. Lancer l'application
Exécutez votre application Java

### 3. Se connecter
- **Email** : admin@nutrilife.com
- **Mot de passe** : admin123
