    🔧 SOLUTION RAPIDE - CAMÉRA NOIRE
═══════════════════════════════════════════════════════════════════════════

PROBLÈME : La caméra reste noire (écran noir au lieu de votre visage)

═══════════════════════════════════════════════════════════════════════════
    ⚡ SOLUTION RAPIDE (3 ÉTAPES)
═══════════════════════════════════════════════════════════════════════════

ÉTAPE 1 : FERMEZ LES APPLICATIONS
──────────────────────────────────────────────────────────────────────────
Fermez TOUTES ces applications si elles sont ouvertes :
    • Skype
    • Microsoft Teams
    • Zoom
    • Discord
    • OBS Studio
    • Toute autre application de visioconférence

💡 Ces applications bloquent l'accès à la caméra


ÉTAPE 2 : LANCEZ LE DIAGNOSTIC
──────────────────────────────────────────────────────────────────────────
Double-cliquez sur :

    📁 DIAGNOSTIC_CAMERA_COMPLET.bat

Lisez les résultats :
    ✓ = OK
    ✗ = Problème
    ⚠️ = Attention

Si vous voyez "✗ OpenCV manquant" :
    → Double-cliquez sur : INSTALL_PROFESSIONAL_FACE_ID.bat


ÉTAPE 3 : TESTEZ LA CAMÉRA
──────────────────────────────────────────────────────────────────────────
Double-cliquez sur :

    📁 FIX_CAMERA_NOW.bat

Vous devriez voir :
    • Votre visage en temps réel
    • Une fenêtre avec le flux vidéo

Si ça fonctionne ici mais pas dans l'application Java :
    → Le problème vient de l'application Java, pas de la caméra

═══════════════════════════════════════════════════════════════════════════
    🔍 DIAGNOSTIC AVANCÉ
═══════════════════════════════════════════════════════════════════════════

SI LA CAMÉRA RESTE NOIRE APRÈS LES 3 ÉTAPES :

1. VÉRIFIEZ LES PERMISSIONS WINDOWS
──────────────────────────────────────────────────────────────────────────
   a) Ouvrez : Paramètres Windows (touche Windows + I)
   b) Allez dans : Confidentialité et sécurité → Caméra
   c) Activez :
      ☑ "Autoriser les applications à accéder à votre caméra"
      ☑ "Autoriser les applications de bureau à accéder à votre caméra"


2. TESTEZ AVEC UN SCRIPT PYTHON SIMPLE
──────────────────────────────────────────────────────────────────────────
   Ouvrez un terminal dans le dossier projetJAV et tapez :
   
       python test_camera_simple.py
   
   Une fenêtre devrait s'ouvrir avec votre visage.
   
   Si ça fonctionne :
       ✅ La caméra marche, le problème est ailleurs
   
   Si ça ne fonctionne pas :
       ❌ Problème matériel ou pilote


3. VÉRIFIEZ LE PILOTE DE LA CAMÉRA
──────────────────────────────────────────────────────────────────────────
   a) Ouvrez : Gestionnaire de périphériques
   b) Développez : Caméras ou Périphériques d'acquisition d'images
   c) Clic droit sur votre caméra → Propriétés
   d) Vérifiez : "Cet appareil fonctionne correctement"
   
   Si vous voyez une erreur :
       → Clic droit → Mettre à jour le pilote


4. REDÉMARREZ VOTRE ORDINATEUR
──────────────────────────────────────────────────────────────────────────
   Parfois un simple redémarrage résout le problème.


5. VÉRIFIEZ L'ÉCLAIRAGE
──────────────────────────────────────────────────────────────────────────
   • La pièce est-elle trop sombre ?
   • Y a-t-il de la lumière devant vous (pas derrière) ?
   • La caméra n'est pas couverte physiquement ?

═══════════════════════════════════════════════════════════════════════════
    🐛 PROBLÈMES SPÉCIFIQUES
═══════════════════════════════════════════════════════════════════════════

PROBLÈME : "Python n'est pas installé"
──────────────────────────────────────────────────────────────────────────
SOLUTION :
    1. Téléchargez Python : https://www.python.org/downloads/
    2. Pendant l'installation, cochez "Add Python to PATH"
    3. Relancez INSTALL_PROFESSIONAL_FACE_ID.bat


PROBLÈME : "OpenCV manquant"
──────────────────────────────────────────────────────────────────────────
SOLUTION :
    Ouvrez un terminal et tapez :
        pip install opencv-python


PROBLÈME : "Port 7654 déjà utilisé"
──────────────────────────────────────────────────────────────────────────
SOLUTION :
    Un serveur caméra est déjà lancé. Tuez le processus :
        taskkill /F /IM python.exe


PROBLÈME : La caméra fonctionne dans le test mais pas dans l'app Java
──────────────────────────────────────────────────────────────────────────
SOLUTION :
    1. Vérifiez que le serveur Python démarre :
       Regardez les logs dans la console Java
       Vous devriez voir : "[CamServer] Ready on 127.0.0.1:7654"
    
    2. Si vous ne voyez pas ce message :
       → Le serveur Python ne démarre pas
       → Vérifiez que camera_server.py ou camera_server_pro.py existe
    
    3. Relancez l'application Java


PROBLÈME : "Camera unavailable after retries"
──────────────────────────────────────────────────────────────────────────
SOLUTION :
    1. Fermez l'application Java
    2. Tuez tous les processus Python : taskkill /F /IM python.exe
    3. Attendez 5 secondes
    4. Relancez l'application


PROBLÈME : Image floue ou sombre
──────────────────────────────────────────────────────────────────────────
SOLUTION :
    • Allumez une lumière devant vous
    • Nettoyez l'objectif de la caméra
    • Ajustez la luminosité de votre écran
    • Éloignez-vous des fenêtres (contre-jour)
