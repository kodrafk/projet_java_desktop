## 🔧 Troubleshooting

### Problem: "Python not found"
**Solution:** Install Python from python.org (check "Add to PATH")

### Problem: "pip install failed"
**Solution:** Run as Administrator, or manually run:
```bash
pip install deepface tf-keras opencv-python retina-face
```

### Problem: "Camera frozen"
**Solution:** 
1. Close Teams/Zoom/Chrome
2. Run: `FIX_CAMERA_NOW.bat`
3. Restart application

### Problem: "Face not detected"
**Solution:**
- Improve lighting (face a window/lamp)
- Look directly at camera
- Distance: 30-50cm
- Remove glasses/mask

### Problem: "Still not recognizing me"
**Solution:**
1. Delete Face ID: Admin → User Management → Edit → Delete Face ID
2. Re-enroll with GOOD lighting
3. Test again
