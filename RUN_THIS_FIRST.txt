│ 🐛 EN CAS DE PROBLÈME                                                     │
└───────────────────────────────────────────────────────────────────────────┘

❌ Problème : Tables non créées
   → Exécutez manuellement CREATE_ANOMALY_DETECTION_TABLES.sql

❌ Problème : Erreur de compilation
   → Vérifiez que tous les fichiers Java sont présents
   → Exécutez : mvn clean compile

❌ Problème : Dashboard vide
   → Cliquez sur "🚀 Lancer Détection"
   → Vérifiez que des utilisateurs ont des weight_logs

❌ Problème : Menu invisible
   → Vérifiez que vous êtes connecté en ADMIN
   → Vérifiez que admin_layout.fxml a été modifié

❌ Problème : Aucune anomalie
   → Exécutez TEST_ANOMALY_DETECTION.bat
   → Ou insérez manuellement les données de test
